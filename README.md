# low-level-lab 🐧

> "The kernel is not a black box; it's just memory that hasn't been mapped yet."

### 🔬 Focus Areas
- **Binary Analysis:** Understanding ELF structures and headers.
- **Memory Forensics:** Investigating `/proc/kcore` and system RAM.
- **Kernel Internals:** Exploring syscalls and kernel modules.

### 🛠 Tools & Experiments
- `elf_check.c`: A small utility to validate ELF magic numbers.
- `mem_dump.sh`: Basic script to interact with process memory.
